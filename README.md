# 💼 Business Chatbot

An intelligent business chatbot built with **React** and **Express.js** that provides instant responses to frequently asked customer questions. The application features persistent chat history, animated streaming responses, a professional business interface, and a configurable knowledge base for small businesses.

## 🚀 Getting Started

### Clone the repository

```bash
git clone <repository-url>
```

### Install frontend dependencies

```bash
npm install
```

### Install backend dependencies

```bash
cd server
npm install
```

### Start the backend

```bash
npm start
```

### Start the frontend

```bash
npm run dev
```
